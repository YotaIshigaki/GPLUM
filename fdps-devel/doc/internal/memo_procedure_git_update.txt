現状、Githubの更新は以下の手順で行うことになっています：

[1] Github更新者が、適当な場所で、SVNの
upload_tools/upload_to_github.sh
を実行する。実行に際しては、シェルスクリプト内の変数
FDPS_SVN_USER
を適宜変更する。
このシェルスクリプト内で、テストスクリプトも実行されます。

[2] 特に問題がなければ、上記シェルスクリプトで作成されるディレクトリFDPSに移動し、
$ git commit 
$ git push
を実行する。

[3] ウェブブラウザを開き、FDPSのGithubのページでタグ作成・編集する。
